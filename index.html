Here's a cool project I've worked on!
Cloud Security Response Validation Exercises

I developed tests known as Cloud Security Response Validation exercises to evaluate Security Incident Response Team’s (SIRT) response to cloud security incidents and vulnerabilities from incident creation to closure. I utilized AWS to imitate known vulnerabilities and security breaches to evaluate SIRT on its ability to resolve cloud security incidents and address improper cloud security hygiene. I designed functional/nonfunctional requirements, report templates, and documentation for the exercises, obtained approval of each from Cloud Security and Cyber Security Operations Center management. I crafted and presented post -incident report detailing SIRT’s interactions during exercise, logging metrics, timeline of events, efficacy of solution, best possible solution, and areas of improvement in response.
